# Landing Page Larissa Botelho

Este projeto é uma Landing Page desenvolvida com **HTML, CSS e JavaScript**, com foco em apresentar uma interface simples, moderna e responsiva.

### 📌 Sobre o projeto

A Landing Page foi criada com o objetivo de apresentar os serviços oferecidos por uma advogada, destacando sua área de atuação, seus conhecimentos profissionais e sua forma de atendimento.
